Privacy Policy / 개인정보처리방침

This app ("we", "our", or "us") respects your privacy and is committed to protecting the information you share with us.
본 앱(이하 "당사")는 사용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수하고 있습니다.

1. Data Collection / 수집하는 정보

We use Google Firebase Analytics to collect anonymous usage data.
당사는 Firebase Analytics를 사용하여 익명의 앱 사용 데이터를 수집합니다.

The collected data includes:
수집되는 정보는 다음과 같습니다:

  • Device information (model, OS version) / 기기 정보 (모델, 운영체제 버전)
  • App usage behavior / 앱 사용 행동 정보
  • Crash logs / 앱 크래시 로그

No personally identifiable information (PII) is collected.
이름, 이메일 등 개인을 식별할 수 있는 정보는 수집되지 않습니다.

2. Use of Data / 수집한 정보의 사용 목적

The data is used solely for improving app performance and user experience.
수집된 정보는 앱 성능 개선 및 사용자 경험 향상을 위해서만 사용됩니다.

3. Data Security / 데이터 보안

All data is encrypted in transit and securely stored on Google Firebase servers.
모든 데이터는 전송 중 암호화되며, Google Firebase 서버에 안전하게 저장됩니다.

4. Data Sharing / 제3자 제공

The data is not shared with any third parties except Google Firebase for analytics purposes.
수집된 데이터는 분석을 위해 Google Firebase 외에 제3자에게 제공되지 않습니다.
